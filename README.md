# Travel Agency Website 🌍✈️

A responsive travel agency website built with HTML and CSS, focused on creating a clean and user-friendly layout.

Live Demo: https://ondrej108.github.io/Travel-Agency-Website/

---

## Features

- Responsive design (mobile, tablet, desktop)
- Multi-section layout (hero, services, trips, reviews)
- Mobile navigation (burger menu)
- Smooth scrolling

---

## Technologies

- HTML5
- CSS3 (Flexbox, Grid)
- CSS Custom Properties
- Media Queries

---

## What I focused on

- Translating design into code
- Building structured layouts
- Writing clean and maintainable CSS
